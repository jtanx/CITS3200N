This is the source code for the mobile application. It is a hybrid mobile app,
written using the Ionic framework (HTML5/AngularJS/CSS). 

The app source files are under the www folder. Knowledge of how AngularJS works
is essential to understand how it runs. Platform specific files (e.g. the XCode
project for iOS deployment) exists in the platforms folder. 

For basic install/run instructions, see INSTALL.txt

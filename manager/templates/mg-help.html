UWA NTPT Help Page

Viewing New Responses

  1. Select the 'Home' tab from dark grey navigation bar at the top of the page.



  2. New responses will be located under the welcome message.



Adding New Users

  1. Select the 'Users' tab from dark grey navigation bar at the top of the page.



  2. Click on the blue plus sign.



  3. Enter the new user's details, and then click the 'Add' button.



Backing Up Data

  1. Select the 'Backup' tab from dark grey navigation bar at the top of the page.



  2. Select the 'Backup Database' option from the drop down menu. Your database will then be downloaded into your computer.



Restoring Data

  1. Select the 'Backup' tab from dark grey navigation bar at the top of the page.



  2. Select the 'Restore Database' option from the drop down menu.



  3. Select the 'Browse' button, and choose a previously saved backup of your data.



  4. Once you have chosen a file, click on the 'Submit' button.



Viewing and Managing Responses

  1. Select the 'Data management' tab from dark grey navigation bar at the top of the page. From the drop down menu, select whichever response category you would like to view.



  2. You can search through responses by using the search options.



  3. You can search delete responses by using the 'Delete' drop down menu.



  4. You can search export responses by using the 'Data' drop down menu.



Editing Your Account Details

  1. Select the far right tab from dark grey navigation bar at the top of the page.



  2. Select the 'Account' option from the drop down menu.



  3. Before editing your account details, you must first insert your current password.



  4. When you have finished updating your details, click on the 'Update' button.



Logging Out

  1. Select the far right tab from dark grey navigation bar at the top of the page.



  2. Select the 'Logout' option from the drop down menu.
